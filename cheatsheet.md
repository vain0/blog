# Hexo Cheatsheet
## Set up

```sh
$ npm install
```

## New Post

```sh
# Create new draft.
$ npx hexo new draft "title"
# Write contents.

# Publish post.
$ npx hexo publish "title"
# Move file into year directory (optional).

# Preview on localhost.
$ npm run serve

# Build.
$ npm run build

# Merge into the master branch. Push to GitHub.
$ git merge ...
$ git push ...
```
